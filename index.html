⚙️ Cara Kerja

4 Langkah Mudah
Mulai Audit Digital

Tidak perlu install software. Tidak perlu server. Cukup Google Account.

1
Copy Google Spreadsheet
Buat salinan template Google Sheets yang sudah disiapkan ke Google Drive Anda. Semua data tersimpan di sana.
2
Deploy Apps Script
Buka Apps Script di dalam Sheet, klik Deploy sebagai Web App. Dapatkan URL aplikasi unik Anda dalam 2 menit.
3
Input Data Master
Isi daftar barang di tab Master Data. Tambahkan akun user tim gudang di tab Users. Sistem siap dipakai.
4
Mulai Audit!
Buka URL di HP tim gudang, login, dan mulai scan & hitung stok. Hasilnya langsung masuk ke Dashboard.
